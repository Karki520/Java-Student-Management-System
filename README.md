# Student Management System

A console-based Java application to manage student records using Core Java concepts.

## Features
- Add new student with roll number, name, course, and marks
- View all student records
- Search student by roll number
- Delete student record
- Input validation using Exception Handling

## Tech Stack
- **Language**: Java
- **Concepts Used**: OOP, ArrayList, Exception Handling, Scanner Class
- **Tools**: VS Code, Git, GitHub

## How to Run
1. Clone the repository:
git clone https://github.com/Karki520/Java-Student-Management-System.git

2. Compile the program:  
 javac StudentManagement.java
3. Run the program:   java StudentManagement

## Project Structure
The application uses two classes:
- `Student`: Model class with private fields and getter methods
- `StudentManagement`: Main class with CRUD operations and menu-driven interface
